# Навигатор Знаний — Telegram-бот для студентов

Бот помогает с расписанием, контактами преподавателей и зачёткой. Работает 24/7 на Render.

## Как запустить локально
1. `pip install -r requirements.txt`
2. Укажи BOT_TOKEN в bot.py
3. `python bot.py`

## Деплой на Render
- Подключи репозиторий к Render
- Добавь переменную BOT_TOKEN = твой_токен
